## Font Size
Use `.fs-{size}` to set the font size.

<div class="example">
  <p o="fs-32">fs-32</p>
  <p o="fs-24">fs-24</p>
  <p o="fs-16">fs-16</p>
</div>

## Font Weight
Use `.fw-{weight}` to set the font size.

<div class="example">
  <p o="fw-900">fw-900</p>
  <p o="fw-700">fw-700</p>
  <p o="fw-400">fw-400</p>
</div>

## Text Color
Use `.color-{color}` to set the text color.

<div class="example">
  <p o="color-red">color-red</p>
  <p o="color-green">color-green</p>
  <p o="color-blue">color-blue</p>
</div>
