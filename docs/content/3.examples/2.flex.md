---
title: Flex
description: Examples of flexbox utility classes and recommended usage.
---

In addition to standard flexbox utilities, following classes are available:
- `.flex-content`: Useful for items that should size to their content, like icons.
  ```
  flex: 0 0 auto
  ```
- `.flex-fill`: Useful for items that should fill the remaining space, but never shrink below their content size.
  ```
  flex: 1 0 auto
  ```
- `.flex-center`
  ```
  display: flex;
  align-items: center;
  justify-content: center;
  ```
