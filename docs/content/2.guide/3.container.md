The preset includes a Container utility that helps to create responsive layouts with consistent padding and maximum widths.

## Usage
To start using it, simply wrap your content using `.container` class.

<div o="container border-1 rounded-8 gap-16 flex flex-col" class="border-default">
  <div class="bg-elevated">normal</div>
  <div class="bg-elevated">normal</div>
</div>

```html
<div class="container">
  <div>element</div>
  <div>element</div>
</div>
```

## Configuration
The Flow Layout can be customized through the configuration file.

```ts [uno.config.ts]
export default defineConfig<Theme>({
  theme: {
    container: {
      maxWidth: { default: 800, sm: 600 }
      padding: 32,
    }
  }
})
```

## Sizes
The container supports multiple max width sizes that can be configured in the configuration file. The sizes can be applied using `data-size` attribute.

<div o="container border-1 rounded-8 gap-16 flex flex-col" class="border-default" data-size="sm">
  <div class="bg-elevated">normal</div>
  <div class="bg-elevated">normal</div>
</div>

```html
<div class="container" data-size="sm">
  <div>element</div>
  <div>element</div>
</div>
```
