---
title: Flow Layout
description: Learn how to use Flow Layout to create dynamic content sections with breakout and full-width elements.
---

The preset includes a Flow Layout. The layout is basically a container that supports also breakout or full width content.

## Usage
To start using it, simply wrap your content using `.flow` class.

The special elements can be marked using `data-flow` attribute. The support values are:
- `breakout`: makes the element breakout of the normal content width, but still respects the page padding.
- `full-width`: makes the element span the full width of the viewport.

<div o="flow border-1 rounded-8 gap-16 py-16" class="border-default">
  <div class="bg-elevated">element</div>
  <div class="bg-elevated">element</div>
  <div data-flow="breakout" class="bg-elevated">breakout element</div>
  <div class="bg-elevated">element</div>
  <div class="bg-elevated">element</div>
  <div data-flow="full-width" class="bg-elevated">full width element</div>
  <div class="bg-elevated">element</div>
</div>

```html
<div class="flow">
  <div>element</div>
  <div>element</div>
  <div data-flow="breakout">breakout element</div>
  <div>element</div>
  <div>element</div>
  <div data-flow="full-width">full width element</div>
  <div>element</div>
</div>
```

## Configuration
The Flow Layout can be customized through the configuration file.

```ts [uno.config.ts]
export default defineConfig<Theme>({
  theme: {
    flow: {
      maxWidth: 800,
      breakoutMaxWidth: 1200,
      padding: 32,
    }
  }
})
```
