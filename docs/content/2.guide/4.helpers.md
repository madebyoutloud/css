title: Helpers
description: Helper functions for JavaScript and SCSS
---

The package exports helper functions for both JavaScript and SCSS.

## JavaScript
JavaScript helpers can be imported from `@outloud/css/helpers`.

```ts
import { toRem } from '@outloud/css/helpers'
```

::field-group
  ::field{name="toRem(value, base?)"}
    Converts number value in pixels to rem string.

    ```ts
    toRem(8) // '0.5rem'
    toRem(16) // '1rem'
    ```
  ::

  ::field{name="toSize(value, base?)"}
    Converts number or string value to rem if no unit is defined.

    ```ts
    toSize(16) // '1rem'
    toSize('4px') // '4px'
    toSize('4em') // '4em'
    ```
  ::
::

## SCSS
SCSS helpers can be imported from `@outloud/css/helpers.scss`.

```scss
@use "@outloud/css/helpers.scss" as *;
```

### Methods
::field-group
  ::field{name="rem"}
    Converts number value in pixels to rem string.

    ```scss
    .card {
      padding: rem(8); // '0.5rem'
      margin: rem(16); // '1rem'
    }
    ```
  ::
::
