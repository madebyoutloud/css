title: Grid System
description: Learn how to use grid system to create responsive layouts with ease.
---

The preset includes a grid system that helps you create responsive layouts quickly and easily. The grid system is based on a 12-column layout, allowing you to divide your content into equal or custom-width columns.

## Usage
To start with grid system, simply wrap create grid container using `.columns` class.

You can adjust the width of each column by using the `.col-{n}` class, where `{n}` is a number from 1 to 12 representing the number of columns the element should span.

<div o="bg-stripes columns fs-24 fw-600 rounded-8 color-white">
  <div o="bg-primary rounded-8 p-12 col-4">01</div>
  <div o="bg-primary rounded-8 p-12 col-4">02</div>
  <div o="bg-primary rounded-8 p-12 col-4">03</div>
</div>

```html
<div class="columns">
  <div class="col-4">01</div>
  <div class="col-4">02</div>
  <div class="col-4">03</div>
</div>
```

## Configuration
The grid system can be customized through the configuration file. You can adjust the number of columns and default gutter size.

```ts [uno.config.ts]
export default defineConfig<Theme>({
  theme: {
    grid: {
      columns: 12,
      gap: 16,
    }
  }
})

## Examples

### Alignment
The columns can be aligned using classic grid align classes.

<div o="bg-stripes columns fs-24 fw-600 rounded-8 color-white h-180">
  <div o="bg-primary rounded-8 p-12 col-4 self-start">01</div>
  <div o="bg-primary rounded-8 p-12 col-4 self-center">02</div>
  <div o="bg-primary rounded-8 p-12 col-4 self-end">03</div>
</div>

```html
<div class="columns h-180">
  <div class="col-4 self-start">01</div>
  <div class="col-4 self-center">02</div>
  <div class="col-4 self-end">03</div>
</div>
```

### Offset
Offset `.offset-{n}` classes can be used to create space before a column.

<div o="bg-stripes columns fs-24 fw-600 rounded-8 color-white">
  <div o="bg-primary rounded-8 p-12 col-3">01</div>
  <div o="bg-primary rounded-8 p-12 col-3 offset-10">02</div>
  <div o="bg-primary rounded-8 p-12 col-3 offset-4">03</div>
  <div o="bg-primary rounded-8 p-12 col-3">04</div>
</div>

```html
<div class="columns">
  <div class="col-3">01</div>
  <div class="col-3 offset-10">02</div>
  <div class="col-3 offset-4">03</div>
  <div class="col-3">04</div>
</div>
```

### Breakpoints

```html
<div class="columns">
  <div class="col-12 md:col-6">01</div>

</div>
```
